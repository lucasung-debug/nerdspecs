# NerdSpecs

**[English](README.md)** | **[한국어](README.ko.md)** | **[中文](README.zh.md)**

> 너드 안경을 쓰면 프로젝트가 보입니다.

NerdSpecs는 AI로 만든 프로젝트를 누구나 이해할 수 있는 문서로 바꿔주는 MCP 서버입니다. **모든 AI CLI**에서 작동합니다 — Claude Code, Gemini CLI, Codex, OpenCode, Qwen Code, Cursor, VS Code 등.

**API 키 불필요. 설정 불필요. 추가하고 "이 프로젝트 README 만들어줘"라고 말하면 끝.**

**[라이브 데모](https://lucasung-debug.github.io/nerdspecs/)** · **[npm](https://www.npmjs.com/package/nerdspecs)**

## 빠른 시작

AI CLI 설정에 NerdSpecs를 추가하고, 문서 생성을 요청하세요.

### Claude Code / Gemini CLI / Qwen Code / Cursor

설정 파일(`.mcp.json`, `~/.gemini/settings.json` 등)에 추가:

```json
{
  "mcpServers": {
    "nerdspecs": {
      "command": "npx",
      "args": ["-y", "nerdspecs"]
    }
  }
}
```

### OpenAI Codex CLI

`~/.codex/config.toml`에 추가:

```toml
[mcp_servers.nerdspecs]
command = "npx"
args = ["-y", "nerdspecs"]
```

### VS Code (GitHub Copilot)

`.vscode/mcp.json`에 추가:

```json
{
  "servers": {
    "nerdspecs": {
      "type": "stdio",
      "command": "npx",
      "args": ["-y", "nerdspecs"]
    }
  }
}
```

그 다음:

> "이 프로젝트 분석해서 README랑 랜딩 페이지 만들어줘."

## MCP 도구

NerdSpecs는 AI 어시스턴트가 사용할 수 있는 8개 도구를 제공합니다:

| 도구 | 기능 |
|------|------|
| `analyze_project` | 프로젝트 파일 스캔 — 언어, 프레임워크, 의존성 감지 |
| `generate_readme` | README.md 생성 (EN/KO/ZH 다국어 지원) |
| `generate_landing_page` | 터미널 데모가 포함된 다크 테마 랜딩 페이지 생성 |
| `fetch_repo_metadata` | AI가 설명할 GitHub 레포 정보 가져오기 |
| `save_decision` | 프로젝트 도입/스킵/관찰 결정 기록 |
| `get_project_status` | 현재 설정 및 저장된 동기 확인 |
| `auto_commit_and_push` | 한 번에 커밋 + 푸시 (비개발자용) |
| `suggest_next_steps` | 프로젝트 상태 분석 후 다음 할 일 제안 |

## 작동 방식

```
당신의 AI CLI (Claude/Gemini/Codex 등)
        |
  NerdSpecs가 코드 분석
        |
  AI가 요약 작성
        |
  NerdSpecs가 템플릿 렌더링
        |
  README.md + 랜딩 페이지
```

핵심 차이: **당신의 AI CLI가 곧 LLM**입니다. NerdSpecs는 AI API를 호출하지 않습니다 — 코드 분석과 템플릿 렌더링만 제공합니다. 요약은 CLI가 자체 지능으로 생성합니다.

## 주요 기능

- **모든 MCP 호환 CLI에서 작동** — Claude Code, Gemini, Codex, OpenCode, Qwen Code, Cursor, VS Code
- **API 키 불필요** — CLI에 이미 인증이 되어 있음
- **3개국어 지원** — 영어, 한국어, 중국어 (EN/KO/ZH)
- **다크 테마 랜딩 페이지** — 터미널 데모, 전후 비교, 기능 카드
- **이미지 슬롯** — Gemini MCP / Pencil / 디자인 도구용 `hero_image_url`, `screenshots`
- **비개발자 친화** — "올려줘" 한마디면 자동 커밋+푸시
- **스마트 제안** — 다음에 뭘 할지 알려줌 (README 없음? 커밋 안 됨?)
- **프로젝트 기억** — 세션 간 답변을 기억

## 이미지 연동

NerdSpecs는 이미지를 생성하지 않습니다 — 다른 도구가 채울 수 있는 슬롯을 제공합니다:

```
AI가 판단:
  ├─ Gemini MCP 있으면 → 히어로 이미지 생성 → hero_image_url
  ├─ Pencil MCP 있으면 → 스크린샷 디자인 → screenshots[]
  ├─ 디자인 스킬 있으면 → 비주얼 생성 → URL 전달
  └─ 아무것도 없으면 → NerdSpecs 터미널 데모 UI (기본값)
```

## 독립 실행 CLI

NerdSpecs는 독립 실행 CLI로도 사용할 수 있습니다:

```bash
npx nerdspecs
```

| 명령어 | 기능 |
|--------|------|
| `nerdspecs write` | README + 랜딩 페이지 생성 (API 키 필요) |
| `nerdspecs read <url>` | GitHub 프로젝트 설명 |
| `nerdspecs think <url>` | 프로젝트 도입 결정 기록 |
| `nerdspecs status` | 설정 표시 |
| `nerdspecs config` | 설정 관리 |
| `nerdspecs memory` | 저장된 데이터 확인/삭제 |
