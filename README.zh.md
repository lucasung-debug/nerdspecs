# NerdSpecs

**[English](README.md)** | **[한국어](README.ko.md)** | **[中文](README.zh.md)**

> 戴上书呆子眼镜，项目就变得清晰可见。

NerdSpecs 是一个 MCP 服务器，可以将你用 AI 构建的项目转换为人人可读的文档。适用于**所有 AI CLI** — Claude Code、Gemini CLI、Codex、OpenCode、Qwen Code、Cursor、VS Code 等。

**无需 API 密钥。无需配置。添加后说一句"给这个项目生成 README"就行了。**

**[在线演示](https://lucasung-debug.github.io/nerdspecs/)** · **[npm](https://www.npmjs.com/package/nerdspecs)**

## 快速开始

将 NerdSpecs 添加到你的 AI CLI 配置中，然后让它生成文档。

### Claude Code / Gemini CLI / Qwen Code / Cursor

添加到配置文件（`.mcp.json`、`~/.gemini/settings.json` 等）：

```json
{
  "mcpServers": {
    "nerdspecs": {
      "command": "npx",
      "args": ["-y", "nerdspecs"]
    }
  }
}
```

### OpenAI Codex CLI

添加到 `~/.codex/config.toml`：

```toml
[mcp_servers.nerdspecs]
command = "npx"
args = ["-y", "nerdspecs"]
```

### VS Code (GitHub Copilot)

添加到 `.vscode/mcp.json`：

```json
{
  "servers": {
    "nerdspecs": {
      "type": "stdio",
      "command": "npx",
      "args": ["-y", "nerdspecs"]
    }
  }
}
```

然后直接说：

> "分析这个项目，生成 README 和着陆页。"

## MCP 工具

NerdSpecs 提供 8 个工具供你的 AI 助手使用：

| 工具 | 功能 |
|------|------|
| `analyze_project` | 扫描项目文件 — 检测语言、框架、依赖 |
| `generate_readme` | 生成 README.md（支持 EN/KO/ZH 多语言） |
| `generate_landing_page` | 生成带终端演示的暗色主题着陆页 |
| `fetch_repo_metadata` | 获取 GitHub 仓库信息供 AI 解释 |
| `save_decision` | 记录项目的采用/跳过/观察决策 |
| `get_project_status` | 查看当前配置和存储的动机 |
| `auto_commit_and_push` | 一步完成提交和推送（面向非开发者） |
| `suggest_next_steps` | 分析项目状态，建议下一步操作 |

## 工作原理

```
你的 AI CLI（Claude/Gemini/Codex 等）
        |
  NerdSpecs 分析你的代码
        |
  AI 撰写摘要
        |
  NerdSpecs 渲染模板
        |
  README.md + 着陆页
```

关键区别：**你的 AI CLI 就是 LLM**。NerdSpecs 不调用任何 AI API — 它只提供代码分析和模板渲染。摘要由你的 CLI 用自身的智能生成。

## 功能特点

- **适用于所有 MCP 兼容 CLI** — Claude Code、Gemini、Codex、OpenCode、Qwen Code、Cursor、VS Code
- **无需 API 密钥** — 你的 CLI 已经完成了认证
- **三语言支持** — 英语、韩语、中文（EN/KO/ZH）
- **暗色主题着陆页** — 终端演示、前后对比、功能卡片
- **图片插槽** — 支持 Gemini MCP / Pencil / 设计工具的 `hero_image_url`、`screenshots`
- **非开发者友好** — 说一句"帮我推送"就自动提交和推送
- **智能建议** — 告诉你接下来该做什么（缺少 README？有未提交的更改？）
- **项目记忆** — 跨会话记住你的回答

## 图片集成

NerdSpecs 不生成图片 — 它提供其他工具可以填充的插槽：

```
AI 判断：
  ├─ 有 Gemini MCP → 生成主图 → hero_image_url
  ├─ 有 Pencil MCP → 设计截图 → screenshots[]
  ├─ 有设计技能 → 创建视觉 → 传递 URL
  └─ 什么都没有 → NerdSpecs 终端演示 UI（默认）
```

## 独立 CLI

NerdSpecs 也可以作为独立 CLI 工具使用：

```bash
npx nerdspecs
```

| 命令 | 功能 |
|------|------|
| `nerdspecs write` | 生成 README + 着陆页（需要 API 密钥） |
| `nerdspecs read <url>` | 解释任何 GitHub 项目 |
| `nerdspecs think <url>` | 记录项目采用决策 |
| `nerdspecs status` | 显示配置 |
| `nerdspecs config` | 管理设置 |
| `nerdspecs memory` | 查看或清除存储数据 |
