Three steps. That's it.

세 단계. 그게 끝.

三步搞定。

1

Add NerdSpecs to your AI tool

AI 도구에 NerdSpecs 추가

将 NerdSpecs 添加到你的 AI 工具

Copy one JSON snippet into your config file. Works with Claude Code, Gemini CLI, Codex, Cursor, VS Code, and more.

설정 파일에 JSON 한 줄 복사. Claude Code, Gemini CLI, Codex, Cursor, VS Code 등 모두 지원.

将一段 JSON 复制到配置文件。支持 Claude Code、Gemini CLI、Codex、Cursor、VS Code 等。

2

Tell your AI what you want

AI에게 원하는 걸 말하기

告诉你的 AI 你想要什么

"Make a README for this project." Your AI analyzes the code, writes a summary, and NerdSpecs turns it into beautiful docs.

"이 프로젝트 README 만들어줘." AI가 코드를 분석하고, 요약을 쓰고, NerdSpecs가 예쁜 문서로 만듭니다.

"给这个项目写个 README。" AI 分析代码、写摘要，NerdSpecs 生成精美文档。

3

Done. Published. Shared.

끝. 배포됨. 공유됨.

完成。已发布。已分享。

README + landing page generated, committed, and pushed to GitHub. Say "push this for me" and it's live.

README + 랜딩 페이지 생성, 커밋, GitHub에 푸시까지. "올려줘" 한마디면 끝.

README + 着陆页生成、提交、推送到 GitHub。说一句"帮我推送"就上线了。
