How it works
작동 방식
如何运作

Three steps. Done.

세 단계. 끝.

三步搞定。

1

Add NerdSpecs to your AI tool

AI 도구에 NerdSpecs 추가

添加到你的 AI 工具

Copy one JSON snippet into your config. Works with Claude Code, Gemini CLI, Codex, Cursor, VS Code — any MCP-compatible tool.

설정 파일에 JSON 한 줄 복사. Claude Code, Gemini CLI, Codex, Cursor, VS Code 등 모든 MCP 호환 도구.

将一段 JSON 复制到配置文件。支持 Claude Code、Gemini CLI、Codex、Cursor、VS Code 等所有 MCP 兼容工具。

2

Ask in plain language

자연어로 요청

用自然语言说

"Make a README for this project." Your AI reads the code, writes the summary, NerdSpecs renders it into beautiful documentation.

"이 프로젝트 README 만들어줘." AI가 코드를 읽고 요약을 쓰고, NerdSpecs가 예쁜 문서로 만듭니다.

"给这个项目写个 README。" AI 读代码、写摘要，NerdSpecs 渲染成精美文档。

3

Done. Live. Shared.

끝. 배포됨. 공유됨.

完成。上线。分享。

README + landing page generated and pushed to GitHub. Just say "push this for me."

README + 랜딩 페이지 생성 후 GitHub에 푸시. "올려줘" 한마디면 끝.

生成 README + 着陆页并推送到 GitHub。说一句"帮我推送"就行。
