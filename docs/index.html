Features

주요 기능

Everything you need to explain your project.

프로젝트를 설명하는 데 필요한 모든 것.

💬

One Question Onboarding

질문 하나로 시작

Just answer "why did you build this?" — NerdSpecs handles the rest.

"이걸 왜 만들었나요?" 하나만 답하면 나머지는 NerdSpecs가 처리합니다.

🌏

Bilingual Output

이중 언어 출력

Generate docs in English, Korean, or both at the same time.

영어, 한국어, 또는 동시에 두 언어로 문서를 생성합니다.

🤖

Multiple LLM Providers

다중 LLM 지원

Claude, OpenAI, or offline mode — no API key required to start.

Claude, OpenAI, 또는 오프라인 모드 — API 키 없이도 시작 가능.

🚀

Landing Page Generator

랜딩 페이지 생성

Static HTML page auto-deployed to GitHub Pages.

GitHub Pages에 자동 배포되는 정적 HTML 페이지.

🧠

Project Memory

프로젝트 기억

Remembers your answers across sessions. No repeated questions.

세션 간 답변을 기억합니다. 같은 질문 반복 없음.

📖

Read Any Project

아무 프로젝트나 읽기

Paste a GitHub URL — get a plain-language explanation.

GitHub URL을 붙여넣으면 쉬운 말로 설명해줍니다.

Commands

명령어

Command명령어 Description설명
nerdspecs write Generate README + landing page for your project 프로젝트의 README + 랜딩 페이지 생성
nerdspecs read <url> Get a plain-language explanation of any GitHub project GitHub 프로젝트를 쉬운 말로 설명
nerdspecs think <url> Record why you're adopting or skipping a project 프로젝트 도입/스킵 이유 기록
nerdspecs status Show current configuration 현재 설정 표시
nerdspecs config Manage language, auto-push, and landing page settings 언어, 자동 푸시, 랜딩 페이지 설정 관리
nerdspecs memory View or clear stored data 저장된 데이터 확인 또는 삭제
